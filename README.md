# LeafRx-A-Deep-Learning-Based-Multi-Class-Plant-Disease-Diagnosis-and-Treatment-Recommendation-System
LeafRx is an AI-powered plant health diagnostic system that combines deep learning–based image classification with an intelligent treatment recommendation engine. The system detects plant diseases from leaf images and provides actionable pesticide or treatment guidance based on the identified condition.
